# SplitSmart

Test commit marker: README single-line update.

SplitSmart is a modern Flutter bill-splitting app for Android and iOS with Firebase-ready architecture, smart settlements, spending insights, dark mode, and UPI payment shortcuts.

## Highlights

- Google + email authentication
- Group-based expense management
- Equal, unequal, and percentage splits
- Optimized settlement algorithm to reduce payment hops
- Firestore-ready real-time sync with offline fallback demo mode
- Receipt attachment support
- Spending charts and category insights
- Push notification service hooks
- UPI `Pay Now` support for India

## Getting Started

1. Install Flutter 3.22+ and Dart 3.3+.
2. Run `flutter pub get`.
3. Create a Firebase project and enable:
   - Firebase Auth (Google + Email/Password)
   - Cloud Firestore
   - Firebase Storage
   - Firebase Cloud Messaging
4. Replace the placeholder values in `lib/firebase_options.dart` with your project values.
5. For Google Sign-In, add platform-specific setup files generated by `flutterfire configure`.
6. Run the app with `flutter run`.

## Android setup

Android platform scaffolding now lives under `android/`.

Before building Android:

1. Copy `android/local.properties.example` to `android/local.properties`
2. Download Firebase `google-services.json` into `android/app/google-services.json`
3. Make sure your Firebase Android package matches `com.smart.expensetracker`
4. Generate the Gradle wrapper locally with `cd android && gradle wrapper`
5. For release builds, create `android/app/upload-keystore.jks`
6. Copy `android/key.properties.example` to `android/key.properties`
7. Build with `flutter build appbundle --release`

Detailed release notes are in `docs/android_release.md`.
